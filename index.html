# vue-cytoscape

# What is this

This is a vue wrapper for cytoscape.js.

# Why

We all love vue for creating user interfaces and cytoscape.js as a graph visualization library. Wouldn't be nice if we can create a reusablevue component that exposes cytoscape.js api? This is what this plugin is about.
